# Sistema de Votação

Este é um sistema de votação simples desenvolvido em Node.js com MySQL e frontend básico em HTML/CSS/JS.

## Funcionalidades
- CRUD de Enquetes
- Cadastro dinâmico de opções
- Votação com validação de período ativo
- Resultados em tempo real

## Requisitos
- Node.js
- MySQL
- Navegador Web
